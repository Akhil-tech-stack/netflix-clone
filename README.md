## 🟣 About

A pixel-focused frontend clone of Netflix, built to practice real-world layout techniques — responsive grids, hero banners, and browsing interfaces — using pure HTML, CSS, and JavaScript with no frameworks.

---

## 🟣 Features

- 🎬 Netflix-style landing page with hero banner
- 📱 Fully responsive design across mobile, tablet, and desktop
- 🎞️ Browse movies and TV shows in a scrollable row layout
- 🖥️ Modern UI closely replicating Netflix's design language
- ⚡ Lightweight — built with vanilla HTML/CSS/JS, no dependencies

---

## 🟣 Tech Stack

<div align="center">

<img src="https://skillicons.dev/icons?i=html,css,js" />

- **Structure:** HTML5 (semantic markup)
- **Styling:** CSS3 (Grid & Flexbox for responsive layouts)
- **Interactivity:** Vanilla JavaScript (ES6)

## 🟣 Getting Started

Clone the repository:

```bash
git clone https://github.com/Akhil-tech-stack/netflix-clone.git
cd netflix-clone
```

Simply open `index.html` in your browser — no build tools or dependencies required.

## 🟣 Project Structure

```
netflix-clone/
├── index.html      # Main HTML structure
├── style.css        # Styling and responsive layout

└── assets/           # Images and media
```
